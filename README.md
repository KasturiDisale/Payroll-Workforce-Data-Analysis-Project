# Payroll-Workforce-Data-Analysis-Project

Project Title: Payroll Compensation Analysis

Description:
This project analyzes workforce shift and compensation data using Excel. The goal was to calculate employee earnings across pay periods while applying compensation rules such as bonuses, pay rate changes, and shift thresholds.

Skills Demonstrated:
* Payroll calculations
* Compensation rule modeling
* Excel data analysis
* Workforce data processing
* Data validation and reconciliation
